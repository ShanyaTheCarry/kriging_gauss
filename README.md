# Kriging Gauss — геостатистическое построение геологических поверхностей

Программный комплекс для автоматизированного построения трёхмерных геологических поверхностей методом кригинга с вариограммой Гаусса. Разработан для инженеров-геологов и проектировщиков, работающих в области геотехники и BIM-моделирования.

## Возможности

- Загрузка данных из Excel (столбцы X, Y, Z)
- Построение экспериментальной и теоретической вариограммы (модель Гаусса)
- Ручная корректировка параметров вариограммы (range, sill, nugget)
- Интерполяция методом обычного кригинга (Ordinary Kriging)
- Визуализация результатов: карта изополей, трёхмерная поверхность
- Сохранение результатов: экспорт в Excel, экспорт изолиний в DXF
- Экспорт трёхмерной поверхности в форматы IGES и STEP
- Автоматическая и ручная конвертация IGES и STEP в формат X_T (Parasolid) с использованием утилиты CrossManager

## Требования

### Для запуска из исходного кода

- Python 3.8 или выше
- Установленные зависимости из раздела "Установка"

### Для работы конвертации в X_T

Формат X_T (Parasolid) является проприетарным и требует использования сторонней утилиты для генерации. Для конвертации IGES и STEP файлов в X_T необходим файл CrossManager.exe, который должен находиться в той же папке, что и скрипт kriging_gauss.py или собранный исполняемый файл.

Если CrossManager.exe отсутствует, программа сохраняет полную работоспособность всех функций, кроме конвертации в X_T. Остальные возможности (кригинг, визуализация, экспорт в Excel, DXF, IGES, STEP) остаются доступными.

## Быстрый старт

### Требования

- Установленный Conda (Miniconda или Anaconda)
- Для полной функциональности конвертации в X_T необходим файл CrossManager.exe в той же папке, что и программа

```bash

cd kriging_gauss
conda env create -f environment.yml
